# Torloris
Slowloris attack over tor network.

Kills apache 1.X with 128, apache 2.X or IIS with 256 threads.

**Not effective on nginx servers.**

Cloning from the repository:
```
 git clone https://github.com/Nemesis0U/Torloris.git
```
Running:
```
python torloris.py
```
